# Music-Player-UI
🎵 Music Player UI
A sleek and responsive Music Player UI built using TypeScript, HTML, CSS, and JavaScript. This player supports features like play/pause, seek bar, and playlist selection.

# 🚀 Features
🔊 Play / Pause music
🎚️ Seek bar to scrub through the track
📃 Playlist support
🎨 Modern responsive UI
⚙️ Built with TypeScript

# 🛠️ Tech Stack
TypeScript
HTML
Tailwind/SCSS 
JavaScript 
React or Vite 

# 📦 Installation & Setup
# Clone the repo
git clone 
cd music-player-ui

# Install dependencies
npm install

# Start the development server
npm run dev
